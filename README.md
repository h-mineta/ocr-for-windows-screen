# OCR for Windows Screen 
Windows画面上のテキストをOCRで取り込むPython

## Install

Python 3.12

Tesseract v5.3.3
https://github.com/UB-Mannheim/tesseract/wiki

```powershell
pip.exe install -r .\requirements.txt
```

## Start
```powershell
python3.exe .\main.py
```
